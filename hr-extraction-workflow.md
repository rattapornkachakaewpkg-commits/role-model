## 📋 รายละเอียดแต่ละขั้น

### ขั้น 1: คัดเลือกเคส (Triage) — 🎯 ทักษะ **#1: Observation**

| หัวข้อ | รายละเอียด |
|---|---|
| **HR ทำอะไร** | รับเคสจากหน้างาน → กรองว่า "เคสนี้มี Role Model ไหม?" |
| **เกณฑ์คัดเข้า** | • เคสที่คนทำ "เกินหน้าที่" <br>• มี "ต้นทุนที่จ่ายเอง" (เวลา/เงิน/ความเสี่ยง) <br>• มี "ผลลัพธ์ที่จับต้องได้" |
| **ตัวอย่าง** | เคสสมาชิก ทีม KBB → ✅ ผ่าน (ลงพื้นเอง + จ่ายต้นทุน + เกิดกองทุน AAM take care) |
| **เครื่องมือ** | Checklist 5 ข้อ (Behavior Indicators) |

**💡 ทักษะ HR:** *ความสามารถสังเกต + แยกแยะ "เคสธรรมดา" vs "เคสที่ซ่อน Role Model"*

---

### ขั้น 2: สัมภาษณ์ STAR (BEI) — 🎤 ทักษะ **#2: Interviewing**

| หัวข้อ | รายละเอียด |
|---|---|
| **HR ทำอะไร** | นัด Role Model สัมภาษณ์ 1-on-1 ใช้คำถาม STAR 4 ขั้น |
| **คำถามหลัก** | **S** — สถานการณ์ยากลำบาก? <br>**T** — หน้าที่คืออะไร ใครสั่ง? <br>**A** — ทำอะไร ทำไม ต้นทุน? <br>**R** — ผลเป็นยังไง? |
| **ตัวอย่าง** | "สมาชิก ทีม KBB — ตอนโทรไม่ติด คิดอะไร? ทำไมถึงลงพื้นเอง?" |
| **เครื่องมือ** | คำถาม STAR 4 ข้อ + คำถาม Follow-up |

**💡 ทักษะ HR:** *ถามเจาะลึกโดยไม่ชี้นำ + ฟัง "ความเงียบ" + จับ "ต้นทุนที่จ่ายเอง"*

---

### ขั้น 3: แมป 3 มิติค่านิยม — 🧭 ทักษะ **#3: Knowledge Management**

| หัวข้อ | รายละเอียด |
|---|---|
| **HR ทำอะไร** | นำคำให้การจากขั้น 2 → แมปเข้า 3 มิติ |
| **3 มิติ** | 🟢 เจ้าของลูกค้า · 🟡 อัตโนมัตในการรับใช้ · 🔵 Unique ในการให้บริการ |
| **ตัวอย่าง** | "โทรไม่ติด → ไปหาเอง" → 🟢 "ลูกค้าเป็นของฉัน ไม่ใช่ของหน้าที่" |
| **เครื่องมือ** | Value Framework 3 มิติ + ตาราง Mapping |

**💡 ทักษะ HR:** *แยก "พฤติกรรม" ออกจาก "ค่านิยมที่ซ่อนอยู่"* — เปลี่ยน tacit → explicit

---

### ขั้น 4: Self-Reflection — 🪞 ทักษะ **#4: Coaching**

| หัวข้อ | รายละเอียด |
|---|---|
| **HR ทำอะไร** | ถาม Role Model ให้ "ยกเคสตัวเอง" 3 คำถาม |
| **คำถาม** | • เคสภูมิใจสุด 6 เดือน? <br>• เคสที่ "ไม่มีใครเห็น" แต่ทำ? <br>• สอนน้องใหม่เรื่องอะไรก่อน? |
| **ตัวอย่าง** | สมาชิก ทีม KBB: "เคสลุงป่วย → ไม่มี KPI แต่ยังไปเยี่ยมต่อ" |
| **เครื่องมือ** | คำถาม Self-Reflection 3 ข้อ + Active Listening |

**💡 ทักษะ HR:** *โค้ชให้คน "เห็นตัวเอง" โดยไม่ตัดสิน* — ดึงความภูมิใจออกมา

---

### ขั้น 5: ให้คะแนน (Scoring) — 📊 ทักษะ **#5: Assessment**

| หัวข้อ | รายละเอียด |
|---|---|
| **HR ทำอะไร** | ให้คะแนน Role Model ตาม Rubric 4 ระดับ |
| **Rubric** | 9-10 ต้นแบบ · 7-8 กำลังพัฒนา · 5-6 ทำตามหน้าที่ดี · 1-4 ต้องพัฒนา |
| **ตัวอย่าง** | สมาชิก ทีม KBB: 9-10 (สร้างกองทุน AAM take care + ต้นทุนสูง) |
| **เครื่องมือ** | Scoring Rubric 1 หน้า + Multi-rater (HR + หัวหน้า) |

**💡 ทักษะ HR:** *วัดผลด้วยเกณฑ์ชัดเจน ไม่ bias* + แยก "ดี" vs "ต้นแบบ"

---

### ขั้น 6: ปิดเคส + ถ่ายทอด — 🎁 ทักษะ **#6: Learning & Development**

| หัวข้อ | รายละเอียด |
|---|---|
| **HR ทำอะไร** | ปิดเคส + ผลิต 4 ชิ้น + ถ่ายทอดสู่คนอื่น |
| **4 Outputs** | 📖 Story Script 5 นาที <br>❓ Guide คำถาม <br>📊 Behavior Indicators <br>🏆 Recognition Criteria |
| **ตัวอย่าง** | เคสสมาชิก ทีม KBB → 4 ชิ้น live ที่ `index.html` |
| **เครื่องมือ** | Template 4 ชิ้น + ช่องทางเผยแพร่ (Live site / ห้องเรียน / Townhall) |

**💡 ทักษะ HR:** *ทำให้ความรู้ "หมุนเวียน" — ไม่หยุดอยู่ที่คนเดียว*

---

### 🆕 ขั้น 7: Anti-fraud Audit — 🛡️ ทักษะ **#7: Audit/Anti-fraud** *(เพิ่มใหม่ 23 ก.ค. 2569)*

> **ที่มา:** จากข่าว Fudan Data Mining (ศ.เซียว หยังหัว) — AI โกงได้, คนก็โกงได้
> HR ต้อง "ตรวจสอบว่า Role Model นั้น 'ทำจริง' ไม่ใช่ 'เล่าสวย'"

| หัวข้อ | รายละเอียด |
|---|---|
| **HR ทำอะไร** | ตรวจสอบ 4 ข้อ ก่อนประกาศ "นี่คือ Role Model" |
| **4 Checklist** | 🔎 **หลักฐาน:** ขอรูปถ่าย/ชื่อลูกค้า/วันที่ <br>🔎 **สามเส้า:** ถามหัวหน้า/เพื่ออ่อน/ลูกค้า ยืนยัน <br>🔎 **ย้อนถาม:** "ถ้าเจอเหตุการณ์เดิม จะทำเหมือนเดิมไหม?" <br>🔎 **ตรวจเวลา:** เคสเกิดก่อน/หลังการสัมภาษณ์ |
| **ตัวอย่าง** | เคสสมาชิก ทีม KBB: ✅ มีรูปเยี่ยมลุง + ลุงกล่าวขอบคุณ + กองทุน AAM take care เปิดจริง |
| **เครื่องมือ** | Audit Checklist + Multi-rater (HR + หัวหน้า + เพื่ออ่อน) |

**💡 ทักษะ HR:** *ป้องกัน "Role Model ปลอม" — คนที่เล่าดีแต่ทำไม่เป็น*

**⚠️ เตือน:** แม้ AI ที่ไม่มีจิตใจ/กิเลส ยังโกงได้ — คนยิ่งต้องตรวจสอบให้ดี

## 🆚 เปรียบเทียบ: "HR แบบเดิม" vs "HR ที่สกัด Role Model" vs "HR ที่มี Audit"

| มิติ | HR แบบเดิม | HR ที่สกัด Role Model | HR ที่มี Audit 🆕 |
|---|---|---|---|
| **เป้าหมาย** | คัดคนเก่ง | สกัด "ความเก่ง" ออกมาเป็น Model | **สกัด + ยืนยันว่า "ความเก่ง" นั้นจริง** |
| **เครื่องมือ** | Resume + Interview | เคสจริง + STAR + 3 มิติ + Rubric | + Audit Checklist + Multi-rater |
| **ผลลัพธ์** | คนได้ตำแหน่ง | คนได้ตำแหน่ง + คนอื่นเอาไปเรียนได้ | + **ป้องกัน Role Model ปลอม** |
| **ความรู้** | อยู่ในตัวคน | กลายเป็น Model ที่องค์กรใช้ซ้ำได้ | + **Model ที่ "ตรวจสอบได้"** |
